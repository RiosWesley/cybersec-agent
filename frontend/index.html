Arquitetura PEAS do Agente Inteligente

  • Performance: Classificação precisa de severidade, mapeamento rigoroso MITRE ATT&CK e plano acionável de contenção.
  • Environment (Ambiente): Logs brutos de firewall, IDS/IPS, Apache/Nginx, relatórios de rede e alertas de CVE.
  • Actuators (Atuadores): Interface gráfica do analista SOC, exportação de relatórios estruturados e planos de mitigação.
  • Sensors (Sensores): Entrada de logs textuais, contexto organizacional e histórico de alertas.

Pipeline de Treinamento

O modelo base Qwen-3B-Instruct foi ajustado via **Fine-tuning (QLoRA)** no Google Colab T4 com dados de incidentes reais de cibersegurança do dataset AttackQA. O modelo resultante foi exportado para GGUF em 4 bits, otimizando o consumo de CPU local no Docker.
